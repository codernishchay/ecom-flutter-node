# ecom-flutter-node

| Frontend | Flutter  |
| --- | --- |
| Backend  | Node.js |
| Database | MongoDB |

### Setup

After doing complete setup of flutter on your local. 

### Front-end Setup

```cpp

cd frontend    // attach your android device 
flutter pug get 
flutter run 

```
